===========================================
SETUP STEPS
===========================================

1. **Get Gemini API Key (FREE)**
   
   a. Go to: https://makersuite.google.com/app/apikey
   b. Sign in with Google account
   c. Click "Create API Key"
   d. Copy the API key

2. **Configure API Key**
   
   Open: C:\gamified_learning\ai_service.py
   
   Replace line 4:
   GEMINI_API_KEY = "YOUR_GEMINI_API_KEY_HERE"
   
   With your actual key:
   GEMINI_API_KEY = "AIzaSy..."

3. **Restart Flask App**
   
   Stop the current app (Ctrl+C)
   Run: python app.py

===========================================
TROUBLESHOOTING
===========================================

**Error: "API key not valid"**
- Check if you copied the full API key
- Make sure no extra spaces
- Verify key is active in Google AI Studio

**Error: "Rate limit exceeded"**
- Wait a minute and try again
- Free tier has 60 requests/minute limit

**No response from AI**
- Check internet connection
- Verify API key is configured
- Check console for error messages
