   5. All students get email notification

┌──────────────────────────────────────────────────────────────────┐
│ ⚙️ SETUP (ONE-TIME)                                              │
└──────────────────────────────────────────────────────────────────┘

STEP 1: Get FREE API Key
   → Visit: https://makersuite.google.com/app/apikey
   → Sign in with Google
   → Create API Key
   → Copy key (starts with "AIza...")

STEP 2: Configure
   → Open: ai_service_enhanced.py
   → Line 8: GEMINI_API_KEY = "YOUR_KEY_HERE"
   → Paste your key
   → Save file

STEP 3: Restart
   → Stop Flask app (Ctrl+C)
   → Run: python app.py
   → Done! ✅
