===========================================
SETUP INSTRUCTIONS
===========================================

### STEP 1: Get FREE Gemini API Key

1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the API key (starts with "AIza...")

### STEP 2: Configure API Key

1. Open file: C:\gamified_learning\ai_service_enhanced.py
2. Find line 8:
   GEMINI_API_KEY = "YOUR_GEMINI_API_KEY_HERE"
3. Replace with your key:
   GEMINI_API_KEY = "AIzaSy..."
4. Save the file

### STEP 3: Restart Flask App

1. Stop current app (Ctrl+C in command prompt)
2. Run: python app.py
3. Open: http://localhost:5000

HOW TO USE - TEACHERS
===========================================

### Auto-Generate Quizzes:

1. Login to teacher dashboard
2. Go to "Post Presentation" section
3. Enter presentation title (e.g., "Python Loops")
4. Upload PPT/PDF file
5. ✅ Check "Auto-generate quiz using AI"
6. Click "Post Presentation"

7. AI automatically:
   - Creates 3 quiz questions
   - Adds them to quiz database
   - Sends email to all students
   - Questions appear on student dashboard

### Manual Quiz Creation:

1. Go to "Create Quiz" section
2. Fill in question and options manually
3. Click "Create Quiz"

===========================================
TROUBLESHOOTING
===========================================

Problem: "API key not valid"
Solution: 
- Check you copied full key
- No extra spaces
- Key starts with "AIza"

Problem: "No AI response"
Solution:
- Check internet connection
- Verify API key in ai_service_enhanced.py
- Check console for errors

Problem: "Quiz not generated"
Solution:
- Make sure checkbox is checked
- Presentation title should be descriptive
- Check if API limit reached

Problem: "Rate limit exceeded"
Solution:
- Wait 1 minute
- Free tier: 60 requests/minute
- Try again
